# Remember - Task tracking application
## The Basics 

This is a simple application to add, delete and edit tasks and keep track of daily activities.

## Technology Used

* Language: Kotlin 
* Build : Gradle 
* Android studio


## How to use

* Git clone from command line

* download the zip file
